A statement-style luxury fragrance landing page. Built with Vite, React,
and TypeScript, with motion/react for animation and Lenis for smooth
scroll. The visual language is a pure-black void with a single amber
accent, set in Italiana and DM Sans.

The site is organised as six sections, each with its own motion
signature — a hero that gives way, a notes panel that breathes, an
object that settles, a ritual that unfolds, and a close that recedes.
Restraint is the core design idea: nothing moves unless it earns its
place. Shipped and live.
